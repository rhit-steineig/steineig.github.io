  • Editor Trees

    For the Editor Trees project, I worked with a classmate over the course of a few weeks to create the back end of a text editor using a custom height-balanced tree class in Java. We had to manually implement node additions and deletions, while keeping the tree balanced and updating balance codes, as well as updating the ranks of the nodes.

    Classic Arcade Game (Joust)

    For the Joust project, I collaborated with two classmates to recreate the classic arcade game 'Joust' using Java. We implemented reading level files, complex enemy movement, power-ups, and complex collision logic. We made animations with Java graphics and custom sprites, and used object-oriented design principles to structure our code well.
